# Визуальная история журнала «Огонёк» (1931–2000)

## О проекте

Этот проект представляет собой исследование эволюции визуального языка обложек советского и постсоветского иллюстрированного журнала «Огонёк» на протяжении почти 70 лет его истории (1931–2000).

С помощью методов компьютерного зрения и анализа данных мы отвечаем на вопросы:
- Как менялось количество людей на обложках в разные исторические эпохи?
- Как менялась цветовая гамма, в частности доля красного цвета — одного из ключевых символов советской эпохи?

## Данные

Все изображения для анализа взяты из оцифрованного архива журнала «Огонёк», доступного на сайте публичной библиотеки для некоммерческого использования: [publ.lib.ru](http://publ.lib.ru).

**Полный набор обложек, использованных в исследовании, доступен по ссылке:**
👉 **[Скачать датасет обложек «Огонёк» (Google Drive)](https://drive.google.com/drive/folders/19g0DKNmG0id6yPA7Sn4Vgd4tpXbLOmot?usp=sharing)**

Датасет организован по десятилетиям:
- `1931-1940`
- `1941-1950`
- `1951-1960`
- `1961-1970`
- `1971-1980`
- `1981-1990`
- `1991-2000`

Скачивались через .bat типа:

@echo off
chcp 65001 >nul
title Загрузка журнала "Огонек" 1973-1990

set BASE_URL=https://publ.lib.ru/ARCHIVES/O/''Ogonek''_(jurnal)/
set OUTPUT_DIR=%USERPROFILE%\Desktop\Ogonek_1973-1990

if not exist "%OUTPUT_DIR%" mkdir "%OUTPUT_DIR%"
cd /d "%OUTPUT_DIR%"

echo Скачиваю 1973 год...
curl -L -o "Огонек, 1973, №01.pdf" "%BASE_URL%/%ce%e3%ee%ed%e5%ea,1973,%20b901.pdf"...

## Методы анализа

1.  **Предобработка изображений**:
    - Изменение размера до единой ширины (800px) с сохранением пропорций.
    - Применение сглаживания (GaussianBlur) для уменьшения артефактов сканирования.
    - Нормализация имен файлов для совместимости с различными ОС.

2.  **Компьютерное зрение**:
    - **Детекция лиц**: с помощью каскадов Хаара (`haarcascade_frontalface_default.xml`) для подсчета количества людей на обложке.
    - **Анализ цветовой гаммы**: вычисление доли красного цвета в изображении с использованием цветового пространства HSV, что позволяет более точно выделить красный диапазон, чем пороги в RGB.

3.  **Анализ и визуализация**:
    - Агрегация данных по годам.
    - Построение временных рядов для отслеживания трендов.
    - Интерпретация полученных результатов в историческом контексте.
